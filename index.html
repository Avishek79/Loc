Location Lookup

Upload your Excel sheet, then search by ID or name to get latitude and longitude.

1. Upload your file

Excel (.xlsx, .xls) or CSV. Your file is saved in this browser, so you only need to upload it once.

2. Map your columns

Tell us which column to search and which hold the coordinates.

3. Search

Type an ID to find the coordinates.

Your file is processed and stored entirely in your browser. Nothing is uploaded to a server.
